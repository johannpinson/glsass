## How to use it on a &lt;div&gt;?

To use the `.c-progress` class on another element like a `<div>`, you must add
specific attributes.

First, `role` and `aria-*` attributes are **required** to make it
accessible.<br /> Then, add a inline style to override the custom property
`--c-proress-value` used to set the width in percentage.

The `:indeterminate` state is deducted by the absence of the `aria-valuenow`
attribute.

<Example>
  <div className="o-row">
    <div className="o-col u-6 u-3@md">
      <div
        className="c-progress"
        role="progressbar"
        aria-valuemin="0"
        aria-valuemax="100"
        aria-valuenow="75"
        style={{ '--c-proress-value': '75%' }}
      ></div>
