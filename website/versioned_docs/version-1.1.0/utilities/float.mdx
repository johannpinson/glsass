This utility add three rules to add float on left or right, or remove it, with
responsive variants.

<Example>
  <div className="db">
    <div className="u-fl-l">Left floated content</div>
    <br />
    <div className="u-fl-r">Right floated content</div>
    <br />
    <div className="u-fl-n">No floated content</div>
  </div>
</Example>

```html
<div class="u-fl-l">Left floated content</div>
<div class="u-fl-r">Right floated content</div>
<div class="u-fl-n">No floated content</div>
```
