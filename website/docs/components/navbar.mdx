Navigation menus are complexe, but the navbar component can make it simply to
add an headerwith a brand logo, navigation links and text.

:::info

Of course, this component is lighter and offer less possibilities than others
CSS framework, but it's desired to keep it simple. And generally, if you want
specific layouts, different responsive behavior, or anything else, you may
create your own navigation component, and make it as accessible as possible.

:::

:::danger

Since the component use the "checkbox hack" to manage the open/close state on
mobile, be careful about the `id/for` attributes for the `input` and `label`
element.

:::

:::note

To constrain the max-width or the navbar, you can add a div as the first child
of the `.c-navbar` element with the `.c-navbar-layout` class.<br/>Then, also add
the `.o-layout` object to the new container element.

:::
