## _import-google-fonts()_

<Version version="1.0.0" />

The mixin is used to initialize default font if you use the Google Fonts
service.

```scss
@include fonts.import-google-fonts($families);

// Examples
@include fonts.import-google-fonts("Raleway");
@include fonts.import-google-fonts("Roboto:ital,wght@0,400;0,700;1,400;1,700");
@include fonts.import-google-fonts("Open+Sans:ital0;1" "Lato:wght@400;700");
```
