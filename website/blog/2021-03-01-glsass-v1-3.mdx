This minor version offers two new components and continue to delivers small
improvements and bug fixes since previous patch releases.

<!--truncate-->

## What's new?

### New and updated components

Theses new components contribute to simplify the layout creation with existing
block, like profile page/menu or loading state.

The `button` components receive big improvments for states (disable, active,
focus) and `outline` mode.

### Summary

- new `avatar` component ([docs](/docs/components/avatar))
- new `progress` component ([docs](/docs/components/progress))
- improve `button` variants - outline and states
  ([docs](/docs/components/button)))
- add `graphics.fonts` variables for better headings overrides
- moving `headings` classes from utilities into helpers, and change namespace
  prefix! ([docs](/docs/helpers/headings))

## Documentation

All docs are updated and the [`next` version](/versions) will be focused about
the v1.4.0 to continue to add usefull things inside the framework.
