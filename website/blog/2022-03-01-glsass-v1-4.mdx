This new version doesn't affect mainly the framework, but update the
documentation (docusaurus) and npm dependencies

<!--truncate-->

## What's new?

### Saas peerDependencies

The main change is the update of the minimal version required for the `sass`
package.

### Bugfix

This version also contains a bugfix for the tooltip component where a typo
mismake affect the `aria-label` attribute.

## Documentation

The docs now use the latest beta (16) version of Docusaurus. All previous
versions are kept available on the [version page](/versions).
